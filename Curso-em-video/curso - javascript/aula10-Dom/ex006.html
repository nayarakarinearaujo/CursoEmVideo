1. Usando o atributo onclick diretamente no HTML:

Neste caso, o evento click é associado diretamente ao elemento HTML através do atributo onclick. A função clicar1 é chamada quando o elemento é clicado.

A função clicar1 muda o texto do elemento area1 para "Clicou Aqui" quando ele é clicado.

2. Usando addEventListener no JavaScript:

Neste caso, o evento click (e outros eventos) é associado ao elemento HTML usando JavaScript através do método addEventListener.

Aqui, o elemento area tem três eventos associados:

  • click: Chama a função clicar, que muda o texto para "Clicou!" e o fundo para vermelho.
  • mouseenter: Chama a função entrar, que muda o texto para "Entrou!" quando o mouse entra na área do elemento.
  • mouseout: Chama a função sair, que muda o texto para "Saiu!" e o fundo de volta para verde quando o mouse sai da área do elemento.
